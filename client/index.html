What Friday Can Do

Voice-First Conversation

Real-time bidirectional audio via Gemini Live at 24kHz. Gapless playback through Web Audio API scheduling. Talk naturally — Friday listens and responds in real time.

Screen-Aware Vision

Continuously sees your screen and understands context. Webcam vision is tool-gated for privacy. Friday knows what you're looking at and adapts in real time.

Multi-Agent Team

Atlas (research), Nova (creative), and Cipher (technical) — specialist sub-agents with distinct ElevenLabs voices that execute tasks in the background and report back.

Persistent Memory

Three-tier memory system — short-term, medium-term, and long-term — with automatic consolidation, episodic recall, and semantic search. Friday gets smarter over time.
