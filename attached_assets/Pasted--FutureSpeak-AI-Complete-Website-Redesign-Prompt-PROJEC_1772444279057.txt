# FutureSpeak.AI — Complete Website Redesign Prompt

## PROJECT OVERVIEW

Redesign futurespeak.ai from scratch. This is a personal brand + open-source project + consulting practice site for Stephen C. Webster, an enterprise AI strategist who built the cLaw Specification (an open standard for AI agent governance) and Agent Friday (the reference implementation).

The current site tries to serve enterprise consulting clients and open-source developers on the same page. The redesign separates these audiences while elevating the governance framework as the primary differentiator.

## TECH STACK

- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion for animations
- Deploy-ready for Vercel or Replit
- No CMS — all content is static/hardcoded for now
- Responsive (mobile-first)
- Dark theme primary (space/night sky aesthetic — the brand uses 🌠🔭 star motifs)

## DESIGN DIRECTION

**Aesthetic:** Premium, editorial, slightly cinematic. Think: a beautiful open-source project page meets a high-end consulting firm. Not corporate sterile. Not hacker-dark-mode generic. Sophisticated with warmth.

**Color palette:** Deep navy/black backgrounds. Warm amber/gold accents (the "warm glow" of Agent Friday). Cyan/teal for technical elements. White and soft gray for text. Subtle star-field or particle effects on hero sections — tasteful, not overwhelming.

**Typography:** A clean sans-serif for body (Inter or similar). A slightly distinctive heading font that feels authoritative but not stuffy. Generous whitespace. Large, confident headlines.

**Interactions:** Smooth scroll. Subtle fade-in animations on scroll. Nothing flashy — the content is the star.

## CONTENT NOTES

- All body copy should use natural prose paragraphs, NOT bullet points, unless displaying a specification or technical list
- Tone is confident, warm, intellectually serious — not corporate, not hacker-casual
- Avoid jargon dumps. Every technical claim should be followed by a plain-English explanation
- The word "we" refers to FutureSpeak.AI. "I" is only used on the About page.
- Use "Agentic OS" as the primary descriptor for Agent Friday. NOT "AGI OS." "Agentic OS" is accurate and defensible — it's an operating system built around autonomous agent capabilities. Use it confidently.
- The star motif (🌠🔭) can appear in section headers or as decorative elements, but sparingly — it shouldn't feel like an emoji site
